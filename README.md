# Caesar-cipher

A simple Python program that encrypts and decrypts messages using the classic **Caesar Cipher** technique. Shift letters forward or backward in the alphabet to keep your messages secret!

##  About the Project

The Caesar Cipher is one of the oldest encryption methods. This program allows users to:
-  Encrypt (encode) a message
-  Decrypt (decode) a message
-  Continue encrypting or decrypting multiple messages without restarting the program

##  Features

- Encode and decode text
- Handles letters, spaces, numbers, and punctuation
- Supports any shift value using modulo (`% 26`)
- Simple and interactive command-line interface
- Beginner-friendly Python project
